## About CybertruckSetup

* **Distributive**: Manjaro
* **Window Manager:** i3-gaps
* **Fonts:** Hack and Roboto Condensed
* **Terminal Emulator:** Alacritty
* **Status bars:** Polybar
* **Compositor:** Picom
* **Color Scheme:** Pywal
* **Wallpaper:** Tesla Cybertruck Wallpaper by @merive
