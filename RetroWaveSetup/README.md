## About RetroWaveSetup

* **Distributive**: Manjaro
* **Window Manager:** Gnome
* **Fonts:** Hack, Roboto and JetBrains Mono
* **Terminal Emulator:** Alacritty and Black Box
* **Color Scheme:** Pywal
* **Wallpaper:** RetroWave Wallpaper by @merive
